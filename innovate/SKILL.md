# Innovate

Explore the current project deeply, then answer:

**What's the single smartest and most radically innovative, accretive, useful and compelling addition or change you could make to the project at this point?**

## Process

1. Read CLAUDE.md, README, and key config files to understand the project's purpose and architecture
2. Explore the codebase structure, recent git history, and open issues/TODOs
3. Identify gaps, friction points, and untapped opportunities
4. Propose ONE concrete, actionable change — not a wishlist
5. Explain why it's the highest-leverage move right now
6. Outline implementation steps if the user wants to proceed
